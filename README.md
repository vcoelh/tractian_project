## Features

- Criação de ambiente virtual com `uv`
- Arquivo principal `main.py`

## Pastas de saída:

* Crie a pasta ./output
* Crie a pasta ./logs

## Como rodar
```bash
# Criar ambiente virtual
uv venv .venv

# Ativar o ambiente virtual (Linux/Mac)
source .venv/bin/activate

# Ativar o ambiente virtual (Windows)
.venv\Scripts\activate

# Rodar o projeto
python src/main.py
